# 🌱 THE GREEN HOUSE PWA - VRAIE APPLICATION IPHONE

## ✅ INSTALLATION PWA SUR IPHONE

### 📱 Étapes EXACTES :

1. **Ouvrir Safari** sur votre iPhone
2. **Taper cette URL :** `http://192.168.178.21:3000/app.html`
3. **Attendre** que la page se charge complètement
4. **Appuyer sur le bouton "Installer maintenant"** (apparaît automatiquement)
   **OU**
   Appuyer sur le bouton **Partager** 📤 en bas de Safari
   Puis choisir **"Ajouter à l'écran d'accueil"**
5. **Confirmer l'installation** quand iOS le demande
6. **L'icône apparaît** sur votre écran d'accueil iPhone ! 🎯

### ✅ RÉSULTAT :
- ✅ **Icône native** sur l'écran d'accueil
- ✅ **Se lance comme une vraie app** (sans Safari)
- ✅ **Plein écran** (pas de barre d'adresse)
- ✅ **Contrôles tactiles** optimisés iOS
- ✅ **Connexion MQTT** pour contrôle à distance

## 🌐 UTILISATION À DISTANCE

Une fois l'app installée sur iPhone :

1. **À la maison :** L'app fonctionne via votre WiFi
2. **Dans la rue :** L'app se connecte via MQTT (4G/5G)
3. **Partout dans le monde :** Broker MQTT broker.hivemq.com accessible globalement
